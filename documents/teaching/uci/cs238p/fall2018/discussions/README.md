# Folder Structure:

```
discussions/ (the discussions folder)
-discussion<number>-<name>.odp (If you used openoffice.)
-discussion<number>-<name>.pdf (This is the presentation pdf.)
-discussion<number>-<name>/ (This is directory contains slide source if you used latex.)
--latex files
-discussion<number>-<name>-code/ (This is directory contains any code examples used in the discussion.)
```
